Hackathon Project, Spring 2023

Promptly Figma Plugin

For users to effectively use generative image AI, they must input specific keywords to produce the image in their mind, but figuring this out is an iterative, inefficient, and time-consuming task. As a part of a design hackathon, I reimagined what this process looks like with Promptly, a tool to streamline UI creation on Figma, allowing for designers to work more efficiently and effectively. This project achieved first place with the following case study.
